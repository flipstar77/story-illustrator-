## ğŸ› Troubleshooting

### Python won't start?
```bash
# Check Python is installed:
python --version

# If not found, download from:
# https://www.python.org/downloads/

# Install dependency:
pip install pyautogui
```

### Userscript not working?
1. Make sure Tampermonkey is installed and enabled
2. Check the script is enabled in Tampermonkey dashboard
3. Press Ctrl+Shift+A to show the panel
4. Check browser console (F12) for errors

### AutoHotkey not typing?
1. Make sure you clicked in the input field FIRST
2. Check AHK is running (icon in system tray)
3. Try Ctrl+Alt+I to check status
4. Try setting `useClipboard := true` in the script
